Advanced: Appbar

Appbar

Adds a navigation bar below the topbar. Note that all Appbar links and text are in English regardless of what language YouTube is set to.




Default Appbar Behavior

If the appbar is pinned, it will always be visible. If it's not pinned, it will dynamically show and hide itself (usually becoming visible when you scroll down). Note that this is just to choose the behavior the Appbar uses when YouTube first loads, you can pin and unpin the Appbar at any time with the pin button (on supported layouts).



* Appbar pinning is not supported on layouts before 2014, so this option will have no effect on those layouts.
